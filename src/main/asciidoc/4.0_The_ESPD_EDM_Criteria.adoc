[.text-center]
== IV.The ESPD-EDM Criteria

[.text-left]
The central element of the ESPD is the concept “Criterion”. In general, criteria are used to ask questions and require data that will be used at evaluation time to exclude or select candidate persons, organisations, products, etc.

[.text-left]
The ESPD Exchange Data Model (ESPD-EDM) uses XML data structures to express the questions and ask for (or express) information on the legal exclusion grounds and the  capacities (technical, economic and financial, etc.) to exclude or select Economic Operators as candidates to win a Procurement Procedure.
